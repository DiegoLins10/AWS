# 📘 README – Seção 21: Machine Learning e AI (AWS Cloud Practitioner)

---

## 1️⃣ Amazon Transcribe

Converte fala em texto (Speech-to-Text).

Usos comuns:

* Transcrição de reuniões
* Call centers
* Legendas automáticas

📌 Na prova:
Transcribe = áudio → texto.

---

## 2️⃣ Amazon Rekognition

Serviço de análise de imagens e vídeos.

Pode:

* Detectar objetos
* Reconhecer rostos
* Identificar emoções
* Detectar conteúdo inapropriado

Usos comuns:

* Segurança
* Monitoramento
* Aplicações com imagens

📌 Na prova:
Rekognition = análise de imagem e vídeo.

---

## 3️⃣ Amazon Translate

Traduz texto automaticamente.

Exemplo:

* Traduzir conteúdo de site
* Chat multilíngue

📌 Translate = texto → outro idioma.

---

## 4️⃣ Amazon Polly

Converte texto em fala (Text-to-Speech).

Usos:

* Assistentes virtuais
* Narração automática
* Sistemas de atendimento

📌 Polly = texto → áudio.

---

## 5️⃣ Amazon Comprehend

Serviço de análise de texto usando NLP (Processamento de Linguagem Natural).

Pode identificar:

* Sentimento (positivo/negativo)
* Entidades (nomes, lugares)
* Palavras-chave
* Tópicos

📌 Na prova:
Comprehend = entender texto.

---

## 6️⃣ Amazon Lex

Serviço para criar chatbots e assistentes virtuais.

Baseado na mesma tecnologia da Alexa.

Pode:

* Entender linguagem natural
* Integrar com Lambda
* Criar fluxos conversacionais

📌 Lex = chatbot inteligente.

---

## 7️⃣ Amazon Kendra

Serviço de busca inteligente.

Permite:

* Pesquisa empresarial
* Busca com linguagem natural
* Respostas mais precisas

📌 Kendra = mecanismo de busca inteligente.

---

## 8️⃣ Amazon Textract

Extrai texto e dados de documentos.

Diferente de OCR simples:

* Entende formulários
* Extrai tabelas
* Estrutura dados

Usos:

* Processamento de documentos
* Digitalização automatizada

📌 Textract = extrair dados de documentos.

---

## 9️⃣ Amazon SageMaker

Serviço para construir, treinar e implantar modelos de Machine Learning.

Permite:

* Criar modelos personalizados
* Treinar com seus próprios dados
* Deploy de modelos

É mais técnico que os outros serviços de IA.

📌 Na prova:
SageMaker = criar e treinar modelos próprios de ML.

---

# 🎯 Comparação Rápida

| Serviço     | Função                      |
| ----------- | --------------------------- |
| Transcribe  | Áudio → Texto               |
| Polly       | Texto → Áudio               |
| Translate   | Tradução de texto           |
| Rekognition | Análise de imagem e vídeo   |
| Comprehend  | Análise de texto            |
| Lex         | Chatbot                     |
| Kendra      | Busca inteligente           |
| Textract    | Extrair dados de documentos |

* Rekognition para imagens

* Textract para extrair dados de documentos
* SageMaker para criar modelos personalizados

# 📌 Resumo Final

AWS oferece serviços de IA prontos para uso (pré-treinados) como:

* Reconhecimento de imagem
* Tradução
* Análise de texto
* Chatbots
* Transcrição

E também oferece:

* SageMaker para criação de modelos personalizados.

Para a Cloud Practitioner, o mais importante é saber **qual serviço resolve qual problema**.
